# Loveletter Sky

Romantic animated single-screen web app with a stacked love-card journey.

## Run locally

```bash
python3 -m http.server 8090
```

Open `http://localhost:8090`.

## Customize quickly

- Edit question cards and affirmation lines in `app.js`
- Change visual style in `styles.css`

## Optional music hook

The app tries to play `assets/audio/love-hook.mp3` at the final celebration.

- Add your licensed short clip at that path
- If no file exists, the app plays a soft romantic chime fallback
